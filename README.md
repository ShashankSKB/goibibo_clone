# Goibibo Clone

This project was created using create-react-app and Json server

## How to run app

Clone the app

`git clone <url>`


  
  

### How to run backend

`npm i json-server`
 `json-server --watch db.json --port 3001`
 
 ### How to run frontend

`cd goibibo`
`npm install`
`npm start`
